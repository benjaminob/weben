HTML

It means HyperText Markup Language. It is a marking up language for describing web documents or web pages. The markup language is a set of markup tags and HTML documents are described by HTML tags.

Web documents contain:

1. The Doctype declaration
This defines the documents type to be HTML. The declaration helps the browser to display webpages.
2. HTML tags
It describes the HTML document.
3. Head tag
It provides information about the document.
4. Title tag
It provides the title of the document.
5. Body tag
It describes the visible page content. Only the body area are displayed by the browser.
6. Header
It describes the headings and it ranges from h1 to h6.

HTML tags

These are keywords or tag names surrounded by angle brackets. HTML tags normally comes in pairs, be the first tags and second tags. First, the opening or start tag, second, closing or end tag.

Browser

These are applications used to read HTML documents and display them.

HTML attributes

Attributes provide additional information about an element. Attributes are always specified in the start tag.

Font tags

*Font tags are used to change certain qualities of a block of text on a webpage. It is useful for changing font size, face, color.

Font tags need three attributes:

Size: Default size of HTML is 3.

Representation of font size

  1. Relative value - Positive or negative (+ or -)
  2. Absolute value - uses 1 to 7.

Fonts: Simply refers to the appearance of a text. Fonts actually accepts multiple values.

Lists

All lists can be nested inside each other.

Ordered list
Used when ordering is of importance. It's a container tag.
Unodered list
It is the opposite of ordered list. E.g type: disc, circle, square. (Start attribute does not work).
Definition list
Used in defining a term.
